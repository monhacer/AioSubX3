# 🚀 AioSubX3 (v3)

یک پنل اشتراک مدرن و بهینه‌شده برای موبایل است که تجربه‌ای شبیه به اپلیکیشن‌های بومی (Native) را برای کاربران پنل **3x-ui** فراهم می‌کند.

## ✨ ویژگی‌های کلیدی (نسخه v3)

* 📱 **رابط کاربری شبیه اپلیکیشن:** طراحی کاملاً مدرن با منوی شناور (Bottom Navigation) جهت جابجایی آسان.
* 📈 **بصری‌سازی تعاملی:** دارای نمودار هفتگی مصرف ترافیک (Sparkline Chart) با قابلیت تعامل.
* 📊 **آمار بهبود یافته:** استفاده از حلقه‌های پیشرفت (Progress Ring) برای نمایش بصری وضعیت مصرف.
* 🌙 **پوسته داینامیک:** پشتیبانی هوشمند از حالت تاریک و روشن با افکت‌های شیشه‌ای (Glassmorphism).
* 📱 **اول موبایل:** بهینه‌سازی کامل برای تجربه لمسی و دستگاه‌های موبایل.
* 🌐 **چندزبانه:** پشتیبانی کامل از زبان‌های فارسی و انگلیسی.
* ⚡ **عملکرد بالا:** سرعت بارگذاری بالا با استفاده از `Chart.js` و انیمیشن‌های CSS بهینه.
* ✅ **قابلیت‌های کامل:** قابلیت کپی لینک، نمایش QR کد با حالت بارگذاری (Loading State)، و دسترسی سریع به نرم‌افزارهای کلاینت.
