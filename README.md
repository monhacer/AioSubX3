# 🚀 AioSubX3 (v3)

Modern, app-like, and highly responsive subscription page for **3x-ui v3.3.0+**.

**AioSubX3** یک پنل اشتراک مدرن و بهینه‌شده برای موبایل است که تجربه‌ای شبیه به اپلیکیشن‌های بومی (Native) را برای کاربران پنل **3x-ui** فراهم می‌کند.

## ✨ Key Features (v3 Update)

* 📱 **App-like Interface:** طراحی کاملاً مدرن با منوی شناور (Bottom Navigation) جهت جابجایی آسان.
* 📈 **Interactive Visuals:** دارای نمودار هفتگی مصرف ترافیک (Sparkline Chart) با قابلیت تعامل.
* 📊 **Enhanced Stats:** استفاده از حلقه‌های پیشرفت (Progress Ring) برای نمایش بصری وضعیت مصرف.
* 🌙 **Dynamic Theme:** پشتیبانی هوشمند از حالت تاریک و روشن با افکت‌های شیشه‌ای (Glassmorphism).
* 📱 **Mobile-First:** بهینه‌سازی کامل برای تجربه لمسی و دستگاه‌های موبایل.
* 🌐 **Multilingual:** پشتیبانی کامل از زبان‌های فارسی و انگلیسی.
* ⚡ **Performance:** سرعت بارگذاری بالا با استفاده از `Chart.js` و انیمیشن‌های CSS بهینه.
* ✅ **Full Functionality:** قابلیت کپی لینک، نمایش QR کد با حالت بارگذاری (Loading State)، و دسترسی سریع به نرم‌افزارهای کلاینت.
